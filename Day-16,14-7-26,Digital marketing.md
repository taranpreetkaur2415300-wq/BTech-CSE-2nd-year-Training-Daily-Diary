# Google Analytics
Google Analytics is a free web analytics service developed by Google that helps website and app owners measure and understand user behavior. It collects data about visitors and presents it in reports and dashboards, enabling businesses to improve their online performance.























#### Q1. If the same user is using the same website a second time, will a new session start or will it duplicate the IP address?
Ans. If the user returns within the session timeout (e.g., within 30 minutes of inactivity in Google Analytics), the same session continues.
 #### If the user returns after the session timeout, a new session starts.
#### The IP address is not duplicated. If the user is on the same internet connection, the IP address usually stays the same. Analytics identifies returning users mainly through cookies or client IDs, not just the IP address.

#### Example

#### 10:00 AM – Visit website → Session 1
#### 10:15 AM – Browse more pages → Still Session 1
#### 11:00 AM – Return after 45 minutes → Session 2

#### Q2.What is an IP address, and how do we define/set an IP for our system?

 Ans. An IP (Internet Protocol) address is a unique numerical address assigned to a device on a network. It allows devices to communicate with each other over the internet or a local network.

#### Example

#### IPv4: 192.168.1.100
#### IPv6: 2001:db8::1

#### How is an IP assigned?

#### Automatically (DHCP)- Your router or ISP assigns an IP address automatically (most common).
#### Manually (Static IP)- You can manually configure an IP address in your system's network settings if required.
#### Q3. If I am using the same system on Wi-Fi, will the IP address be the same or different?

 Ans. It depends on the Wi-Fi network:

#### Same system + Same Wi-Fi network-Usually the IP address remains the same (especially the public IP while connected to that router). Your local IP may also stay the same unless the router assigns a new one.
#### Same system + Different Wi-Fi network-The IP address changes because each network has its own IP assignment.
#### Same system + Mobile hotspot- The IP address will also be different from your home or office Wi-Fi.

#### Example

#### Laptop on Home Wi-Fi → Public IP: 103.25.10.5
#### Same laptop on Office Wi-Fi → Public IP: 49.36.120.8
#### Same laptop on Mobile Hotspot → Public IP: 122.176.55.21
