# OPEN GRAPH (OG) TAGS
An Open Graph (OG) tag is a snippet of code in your webpage's HTML that controls how your link appears 
when shared on social media platforms like Facebook, X (Twitter), LinkedIn, and messaging apps.
## Essential OG Tags
### Title (og:title)
The main headline for your shared link. It should be catchy and generally under 60 characters.
### Description (og:description)
A brief summary of your page (around 2-3 sentences) that entices users to click.
### Image (og:image)
The URL of the image that will display alongside your link. This is the most visual element of your share.
### URL (og:url)
The canonical (permanent) link of the page you are sharing.

# Canonical URL
A canonical URL is the preferred, master version of a webpage that search engines like 
Google use to represent a set of duplicate or nearly identical pages.

# Robot.txt File
A robots.txt file is a plain text file placed in your website's root directory that instructs search engine crawlers (like Googlebot) which pages or sections of your site they are allowed or forbidden to crawl.
## Why Robots.txt Matters for SEO
### Prevents Duplicate Content
You can prevent search engines from indexing slightly different versions of pages, avoiding SEO penalties for identical content.
### Manages Server Load
Blocking resource-heavy areas or stopping bots from crawling your site excessively helps keep your web server running fast and stable.
### AI Bot Control
You can explicitly use this file to allow or disallow AI web scrapers (like OpenAI or Anthropic) from using your content to train models or generate answers.
